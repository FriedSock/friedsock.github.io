Bringing version control insights into Vim

Version control plugin user study

Welcome to the study.

If you are reading this page, you have decided to, or are considering being part of a study for one or both of the Smeargle or CtrlpSimilar plugins for Vim and Git.

Smeargle

Smeargle is a plugin to study how we can radiate version control information to developers in real time. The emphasis of the tool is to change the text background colour to reflect the Age/Author of the code you are reading akin to a Choropleth Map take a look at the project page for usage demos, and installation instructions.

CtrlpSimilar

CtrlpSimilar builds upon the existing ctrlp plugin, a fuzzy logic file searcher. Except it aims to predict which files you would like to open, based on the version control history, and the current coding context. Again, take a look at the project page for usage demos, and installation instructions.

What to do

If you would like to proceed, simply send me an email at jbtwentythree'at'gmail.com (replace 'at' with @) saying with which plugin(s) you are trying out, and you will receive a short questionnaire about your experience after a few days usage period. In the case of CtrlpSimilar I ask that you include a logfile in your response, if you have any concerns about this: consult the project page or send me an email. If you have any trouble during installation from the instructions given, or you have any questions at all, send me an email and I will respond as soon as possible.
